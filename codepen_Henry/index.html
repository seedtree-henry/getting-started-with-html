1. World Wide Web

Web shows any contents using HTML. Web shows not only texts but also images and videos.
Also, web connects web pages using hyperlinks.

Compoents of web are described below.
- HTTP : The main protocol of the web
- Servers : Computers that host the files that make up the web
- Internet : The world's largest computer network
- Brower : A program that runs on your computer to display files found on the web 2. HTML

HTML stands for Hypertect Markup Language. HTML is composed of text contents, markup, references to other documents and links to other pages.
